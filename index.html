System Setup

Mac (OSX) Python Setup

OSX actually comes with a system level python. Most developers leave it alone so nothing gets messed up. They will install the versions they need/want through a variety of ways. The easiest way is through a package manager called Homebrew. This is the way I am going to recommend. This is all done from a command line. There is a link to a youtube video that shows step 2 and 3 at the bottom of this section.

Step 1 - Install xcode

open a terminal and run the following command. This will take some time to download and install.

xcode-select --install

Step 2 - Install Homebrew

In the same terminal run the command

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

Step 2 - Install Python 3

In the same terminal run the command

brew install python3

Sound Maker

Sound Maker is a simple Client/Server Application that was built to be a simple and fun introduction to Python for students.

The client was kept pure python on purpose to try to help get the absolute beginner to the excitement of executing code to make another computer on the network do something.

Running the server requires installing the dependencies in the requirements.txt file.

Using the buttons below students can download the files needed in a simple .zip format. This will reduce the barrier to entry. A link to the git repo is also provided for those with more advanced capabilities.
